# 📝 Simple Todo App (Vanilla JavaScript)

A beginner-friendly Todo application built using **HTML, CSS, and JavaScript**.

This project focuses on understanding **DOM manipulation**, dynamic element creation, and basic event handling.

---

## 🚀 Features

* ➕ Add new todo items
* ❌ Delete existing todos
* 🎨 Basic styling using CSS
* ⚡ Dynamic DOM updates

---

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript (Vanilla JS)

---

## 🧠 Concepts Practiced

* `getElementById`
* `createElement`
* `appendChild`
* `removeChild`
* Event handling
* Basic state management
